# Data collection and processing
<br>
## Dataset
final_dataset.csv - медиатека + разметка (пшенная каша) + данные с сайта новостей Коми
final_two_variants_dataset.csv - медиатека + разметка (пшенная каша) + данные с сайта новостей Коми + дублирован с шифтом языка
## Notebook
Check_sentences.ipynb - файл для проверки данных, есть поиск расхождания строк
Make_a_dataset.ipynb - создание общих датасетов в разделе Dataset
Merge_processed_documents.ipynb - объединение файлов ручной обработки
Parser_administration.ipynb - парсинг сайта новостей Коми и сохранение в файлы, в зависимости от их пригодности
